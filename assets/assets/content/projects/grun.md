## Project Overview

GRUN is a watchOS application designed to enhance running efficiency by keeping runners in Heart Rate Zone 2. It uses personalized music to guide the running pace, providing real-time feedback for a healthier run.

### Key Features
- **Zone 2 Training**: Monitors HealthKit data to keep heart rate in the optimal zone.
- **Music Pacing**: Adjusts or selects music via MusicKit to match the target cadence.
- **Watch First**: Designed primarily for Apple Watch for a phone-free experience.
