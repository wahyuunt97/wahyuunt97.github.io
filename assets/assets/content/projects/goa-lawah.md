## Project Overview

Goa Lawah is an Android-based adventure game created for the Global Game Jam 2017 ('Waves' theme). Players control a bat navigating a mysterious cave that shifts between light and dark, avoiding obstacles in a challenging environment.

### Development Speed
- **72 Hours**: Built entirely from scratch during the hackathon.
- **Solo Dev**: Handled all programming duties independently.
