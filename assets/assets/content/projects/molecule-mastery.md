## Project Overview

Molecule Mastery gamifies chemistry learning by combining the 'Atom' card game with Augmented Reality. Players visualize chemical elements in 3D, making abstract concepts tangible and competitive.

### features
- **AR Visualization**: See atoms and molecules in real-world space.
- **Multiplayer**: Competitive gameplay to motivate learning.
- **Cross-Device**: Optimized for both iPhone and iPad screens.
